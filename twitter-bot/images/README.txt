Place manga/manhwa images here for the bot to attach when posting tweets.

Naming convention:
  <series-id>-<number>.jpg   e.g.  sl-01.jpg  sl-02.jpg  gen-01.jpg

The bot will look for an image matching the tweet ID first (e.g. tweet "sl-01" → "sl-01.jpg"),
then fall back to a series-prefix match (e.g. any "sl-*.jpg"), then post without an image.

Supported formats: .jpg  .jpeg  .png  .gif  .webp

Recommended image sources:
- Official manga/manhwa publisher promotional art
- MangVault screenshots / cover art from the site
- Fan art (make sure you have rights to use)
